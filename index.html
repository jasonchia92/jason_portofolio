    • E-Library

    • View Project

    E-Library :
    A online library query system developed with IBM worklight, supported all platform: web, Windows Phone, iOS, Android and etc.
    The app intents to help people like student to use their nearby library resources with more efficiency. With our app, scan book's ISBN by using camera on smartphones, or input by keyboard, so system will review the book's stat through internet by using FINDBOOK's API, and the whole process will just take around 20second to complete.
    Next, four function was being provide:
    1. User can made a books collection check of nearby library by selecting target library from service list.
    2. User can reserve the book if available on targeted library, or book also can be recommend to target library if the book don't available.
    3. User can check the price of the book through internet, try to find out where is this book has more affordable price.
    4. With member system, user can manage their own books borrow list, date to return books, and etc.

    Please refer to the video for more information.

    System merged with SHU-TE University's Library, service online on 2013.

    Award:
    2013 IBM Mobile First International ICT Innovative Service Contest, Silver.
    2011 CSIE STU Application Service Contest, Silver.

    Construction Engineering :
    A B2C system with local industry participate, supported by Web client and Android, purposely connect house decoration company to their customer, and both project designer and project manager. We provide a platform for customer keeping in touch with decoration company for futher discussion about their project and update immediately with every single person in charge in this project.
